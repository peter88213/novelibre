# ODT-Formatvorlagen im *novelibre*-Manuskript und im endgültigen Dokument

Wenn Sie Ihr Dolumentenlayout selbst gestalten, berücksichtigen Sie bitte die folgenden Formatvorlagen:

## Zeichenformate zur Anwendung im Text

| *Writer* Vorlagenname                  | Beschreibung                                           |
|----------------------------------------|--------------------------------------------------------|
| **Betont**                             | Betonter Text.                                         | 
| **Stark betont**                       | Stark betonter Text.                                   | 

---

## Absatzformate zur Anwendung im Text

| *Writer* Vorlagenname                  | Beschreibung                                           |
|----------------------------------------|--------------------------------------------------------|
| **Zitat**                              | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **Überschrift 5**                      | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **Überschrift 6**                      | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **Überschrift 7**                      | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **Überschrift 8**                      | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **Überschrift 9**                      | Absatz, der sich visuell vom Textkörper unterscheidet. |

---

## Automatisch angewendete Absatzformate

| *Writer* Vorlagenname                  | Beschreibung                                           |
|----------------------------------------|--------------------------------------------------------|
| **Überschrift 1**                      | Teileüberschrift.                                      |
| **Überschrift 2**                      | Kapitelüberschrift.                                    |
| **Überschrift 4**                      | Abschnittstrenner.                                     |
| **Titel**                              | Romantitel.                                            |
| **Untertitel**                         | Autor.                                                 |
| **Motto**                              | Motto.                                                 |
| **Motto-Quelle**                       | Die Quelle des Mottos, falls vorhanden.                |
| **Kapitelanfang**                      | Erster Absatz im Kapitel.                              |
| **Textkörper**                         | Erster Absatz im Abschnitt, falls nicht Kapitelanfang. |
| **Textkörper Einzug**                  | Normaler Absatz.                                       |
